Redis 配置与基础
中文文档：redisdoc.com
docker pull redis     ###安装
redis是个noSQL数据库，默认有16个数据库。redis的五大特性STRING, hash, list, set, zset组成。

SELECT 0    //选择0号数据库
FLUSHDB    //删除数据库
FLUSHALL    //删除所有数据库
STRING 是KEY和VALUE组成，value最大值是：512M

SEXEP key 10 value  //时间是10s
SET key value   //key存在就是修改
GET key
MSET key value age value    //多种值的设置
MGET key value  //多种值获取
DEL key //删除key
HASH：

HSET HASH key value
HGET HASH key
HMSET HASH key1 value1 key2 value2     //设置user
HMGET HASH key value      //多个值获取
HLEN HASH   //获取user的产度
HEXISTS HASH key //name是否存在
HDEL HASH key   //删除
HGETALL HASH //获取所有user
List是个简单字符串列表，元素是有序的，，本质是链表，的数据可以在左或右插入，值可以重复，如果值全移除，对应的键值也消息，

lpush list1 value1 value2  //左边顺序插入
rpush  list1 value1 value2  //右边顺序插入
lrange list1 0 -1 //左读取 0第一个元素
lpop list1  //左边删除一个元素
rpop list1  //右边删除一个元素
del list1   //删除list
Set(集合)是无序的，元素不能重复
sadd email a@qq.com b@qq.com c@qq.com   //添加
smembers email  //取出所有值
sismember email a@qq.com    //判断是否存在
srem email a@qq.com //删除
Go Lang redis数据库
Go的链接Redis库：go get github.com/gomodule/redigo/redis
###链接redis
redis.Dial("tcp","127.0.0.1:6379")  //链接Redis
_,err2 := conn.Do("set","name","haoren")    //写入到Redis
str1,err3 := redis.String(conn.Do("get","name"))    //读取Redis转成string 字符串使用redis.Strings
###链接池用法
var pool *redis.Pool //连接池
pool = &redis.Pool {  //获取结构体地址的
    MaxIdle : 8,  //最大空闲
    MaxActive : 0,  //最大链接数
    IdleTimeout : 100,  //超时时间
    Dial : func()(redis.Conn,error) { //redis链接匿名函数
      return redis.Dial("tcp","127.0.0.1:6379")
    },
  }
}
conn := pool.Get()      //从连接池获取链接
conn.Do("mset","name","qiqi","age","25")    //写入数据到redis
